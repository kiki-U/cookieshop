#   网上蛋糕商城-项目搭建

#### 介绍
网上蛋糕商城项目中主要讲解如何建设B2C的网上购物系统。该项目应满足以下需求。
统一友好的操作界面，具有良好的用户体验。
商品分类详尽，可按不同类别分别查看商品信息。
可通过条幅展示推荐商品。
热销推荐和新品推荐的展示。
用户注册、验证和登录功能。
通过蛋糕名称模糊搜索相关商品。
通过购物车一次购买多件商品。
提供简单的安全模型，用户必须登录后才可以购买商品。
用户选择商品后可以在线提交订单。
用户可以查看自己的订单信息。
设计网站后台，用于管理网站的各项基本数据，包括订单管理、商品管理、用户管理以及商品类目。
系统运行安全稳定且响应及时。
